# Viewing Other Profiles

Every person on Tavrn has a profile that tells you a bit about who they are. Here's how to view them and what you can do from there.

---

## The Mini Profile

The quickest way to learn about someone, just click or tap their username or avatar anywhere in Tavrn.

A **mini profile** is a compact popup that appears in-context. It shows:

- Their **avatar** and **display name**
- Their **username** (`@handle`)
- Their current **status** and **custom status message**
- Their **pronouns** (if they've set them)
- A short version of their **bio**
- Quick action buttons

## The Full Profile

Clicking **View Profile** from the mini profile (or navigating to someone's profile page) shows the full thing:

- **Banner color** across the top
- **Avatar** overlapping the banner
- **Display name** and **username**
- **Pronouns** (if set)
- **Custom status** message
- **Bio / About Me** section
- **Steam connection** (if connected and visible)
- **Cocoa badge** (if they're a Cocoa member)
- **Mutual rooms** (rooms you both belong to)
- **Mutual friends** (friends you share)

---

## Mutual Rooms & Mutual Friends

Shared context is a great icebreaker. When you view someone's profile, Tavrn shows you:

- **Rooms you're both in**, shared communities
- **Friends you have in common**, mutual connections

This info is visible to both parties and helps you understand how you're connected before you start a conversation.

---

## Profile Privacy Matters

What you can see on someone's profile depends on **their privacy settings**. You might see:

- **Full profile**, if they're public or you're friends
- **Limited profile**, if they've restricted visibility to friends only
- **Very minimal info**, if their profile is set to private

Some people you can see the username and avatar of, but not their bio or social links. Respect those boundaries, they're intentional.

## The QR Code (2FA Only)

Tavrn uses QR codes in **Settings → Security & 2FA** for setting up two-factor authentication. This is not a profile-sharing feature, it's specifically for scanning with your authenticator app when enrolling 2FA.

[Setting up 2FA →](/account-and-settings/two-factor-authentication)

## Reporting from a Profile

If someone's profile contains content that violates Tavrn's Community Standards, you can report it directly from the profile page. Look for the **⋯ more options** menu and select **Report**.

<Info>
  See [Reporting Users & Content](/privacy-and-safety/reporting-users-and-content) for how the report process works.
</Info>
