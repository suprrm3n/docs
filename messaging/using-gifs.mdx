# Using GIFs

A well-timed GIF can say everything words can't. Tavrn has a built-in GIF picker powered by NovaGPI so you've got the entire internet's reaction GIF library at your fingertips.

---

## Opening the GIF Picker

The GIF button lives in the message toolbar, look for the **GIF** button or the sparkle/lightning icon next to the emoji picker.

Click it to open the GIF panel.

---

## Finding the Right GIF

<Tabs>
  <Tab title="Search">
    Type any keyword into the search bar at the top of the GIF picker. "excited," "facepalm," "cheering," "cat computer", whatever captures the moment.

    Results pull from NovaGPI's catalog, so you've got a *lot* to work with.
  </Tab>
  <Tab title="Trending">
    The default view shows **trending GIFs**, whatever's popular right now across the internet. Good for memes and timely reactions.
  </Tab>
  <Tab title="Categories">
    Browse by category, emotions, reactions, animals, celebrities, gaming, etc. Great when you have a vibe in mind but not a specific keyword.
  </Tab>
  <Tab title="Favorites">
    Find a GIF you'll use over and over? Mark it as a favorite by clicking the ❤️ icon. Your favorites are saved for quick access next time.
  </Tab>
</Tabs>

---

## Sending a GIF

1. Find your GIF using search, trending, or categories
2. Click it, it sends immediately, no extra confirmation needed

The GIF appears as an inline animated preview in the chat. Everyone can see it playing automatically.

<Tip>
  You can also add a text message alongside a GIF, type your message in the input box before clicking the GIF to send both at once.
</Tip>

---

## GIFs vs. Image Files

There are two ways to share animated content on Tavrn:

| | GIF Picker | Uploading a GIF file |
|---|---|---|
| **Source** | NovaGPI library | Your device |
| **Content** | Internet GIFs | Your own animated content |
| **File size concerns** | None (NovaGPI hosts it) | Up to 25MB |
| **How it displays** | Inline animated preview | Inline animated preview |

If you have a GIF saved on your device, upload it like any other image (see [Sharing Files & Media](/messaging/sharing-files-and-media)). For internet GIFs, always use the picker.

## GIF Etiquette (Unwritten Rules)

These aren't official rules, just vibes from communities that work well:

- **Match the energy**, A huge dramatic GIF in a quiet conversation can feel off. Read the room.
- **Don't flood the chat**, Multiple GIFs in rapid succession can overwhelm a channel.
- **Trending doesn't always mean relevant**, A trending GIF that has nothing to do with the conversation is just noise.
- **Reactions > GIFs for simple acknowledgment**, If you just want to say "lol," an emoji reaction might be cleaner.

## Animated Avatars, Not the Same as GIFs in Chat

Separately from GIFs in messages, **Cocoa members** can use animated GIFs as their profile avatar. This is different from the GIF picker and is set up in your profile settings.

  Learn more about animated avatars in [Animated Avatars](/cocoa-premium/animated-avatars).

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="The GIF I'm looking for isn't showing up">
    Try different keywords, NovaGPI's search can be picky. Also try variations of the phrase or related concepts. If a specific GIF exists but isn't in NovaGPI's library, you won't find it through the picker (but you could upload it as a file if you have it saved).
  </Accordion>
  <Accordion title="Can I disable GIF autoplay?">
    Currently, GIFs in chat autoplay by default. If GIF movement is an issue for you (accessibility, distraction, bandwidth), this is worth flagging as a feature request.
  </Accordion>
  <Accordion title="Can I share GIFs that aren't from NovaGPI?">
    Yes, just paste a direct link to any GIF online. It'll display as a link preview, though it may not animate inline the same way as NovaGPI GIFs. Or save the GIF and upload it as an image file.
  </Accordion>
  <Accordion title="Do GIFs use a lot of data?">
    They can, especially on mobile. If you're on a limited data plan, be aware that GIF-heavy chats will use more bandwidth. Each GIF fetches from NovaGPI's CDN when it's in view.
  </Accordion>
</AccordionGroup>
